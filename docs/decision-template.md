# Decision Log Template

_Use when making significant decisions worth tracking._
_File name: `memory/decisions/YYYY-MM-topic.md`_

---

```markdown
# Decision: [Brief Title]

**Date:** YYYY-MM-DD
**Status:** Proposed | Decided | Implemented | Reverted

## Context
[What's the situation? What problem are we solving?]

## Options Considered

### Option 1: [Name]
- Pros: ...
- Cons: ...

### Option 2: [Name]
- Pros: ...
- Cons: ...

## Decision
[What did we decide? Be specific.]

## Rationale
[Why this option?]

## Trade-offs
[What did we give up? What risks did we accept?]

## Follow-up
[What needs to happen next?]
```

---

## When to use
- Architecture or tool choices
- Process changes
- Strategy shifts
- Anything that might be questioned later

## When NOT to use
- Routine tasks
- Minor preferences
- Obvious fixes
